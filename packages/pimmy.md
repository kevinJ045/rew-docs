# Pimmy
[Pimmy](https://github.com/kevinJ045/rew.pimmy) is a package manager built on top of rew to help simplify the process of installing packages in rew. 

## Installing
You can install Pimmy from `github:kevinJ045/rew.pimmy` using the `rew install` command.
```bash
rew install github:kevinJ045/rew.pimmy
```
Or with rewpkgs
```bash
rew install @rewpkgs/rew.pimmy
```
